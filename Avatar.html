    • Avatar

    An avatar is a personalized graphical illustration that represents a computer user, or a character or alter ego that represents that user.

    Profile Avatar

    Profile Avatars can be created using the class 'avatar-img', inside the img tag.

    For different sizes of avatar, include avatar-small, avatar-medium and avatar-large along with the avatar-img class for small(1.5rem), medium(4rem) and large(5rem) image respectively.

    Square Avatar

    Adding a class 'avatar-square' to existing class, will generate a square shaped avatar. And for different sizes, add the class accordingly.
